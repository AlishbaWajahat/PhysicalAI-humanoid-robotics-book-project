# Data Model: Docusaurus Configuration

**Feature**: 001-docusaurus-setup
**Phase**: Phase 1
**Date**: 2025-12-04

## Configuration Entity

The Docusaurus configuration is defined in `docusaurus.config.js` as a JavaScript object.

### Core Configuration

```javascript
// docusaurus.config.js
module.exports = {
  // Site metadata
  title: 'Physical AI & Humanoid Robotics',
  tagline: 'The Official Panaversity AI-Native Textbook',
  url: 'http://localhost:3000',  // Temporary, localhost only
  baseUrl: '/',

  // Deployment (not used yet)
  organizationName: 'TBD',  // Placeholder
  projectName: 'TBD',       // Placeholder

  // Build behavior
  onBrokenLinks: 'throw',
  onBrokenMarkdownLinks: 'warn',

  // Internationalization (not configured yet)
  i18n: {
    defaultLocale: 'en',
    locales: ['en'],
  },

  // Presets
  presets: [
    [
      '@docusaurus/preset-classic',
      {
        docs: {
          sidebarPath: './sidebars.js',
        },
        blog: false,  // Disabled for textbook
        theme: {
          customCss: './src/css/custom.css',
        },
      },
    ],
  ],

  // Plugins
  plugins: [
    'docusaurus-plugin-sass',
  ],

  // Theme configuration
  themeConfig: {
    navbar: {
      title: 'Physical AI & Humanoid Robotics',
      items: [
        {
          type: 'docSidebar',
          sidebarId: 'tutorialSidebar',
          position: 'left',
          label: 'Tutorial',
        },
      ],
    },
    footer: {
      style: 'dark',
      copyright: `Copyright © ${new Date().getFullYear()} Panaversity`,
    },
  },
};
```

## Entity Fields

### Required Fields
- **title** (string): Site title - "Physical AI & Humanoid Robotics"
- **tagline** (string): Site tagline - "The Official Panaversity AI-Native Textbook"
- **url** (string): Base URL - "http://localhost:3000" (temporary)
- **baseUrl** (string): Base path - "/"

### Preset Configuration
- **@docusaurus/preset-classic**: Default Docusaurus preset
  - docs.sidebarPath: Points to sidebar configuration
  - blog: Disabled (false)
  - theme.customCss: Custom CSS file path

### Plugin Configuration
- **docusaurus-plugin-sass**: SASS/SCSS support
  - No additional configuration required
  - Automatically processes .scss files

## Sidebar Configuration

Defined in `sidebars.js`:

```javascript
module.exports = {
  tutorialSidebar: [
    {
      type: 'autogenerated',
      dirName: '.',
    },
  ],
};
```

## Validation Rules

1. **title**: Non-empty string, 80 characters max
2. **tagline**: Non-empty string, 120 characters max
3. **url**: Valid URL format
4. **baseUrl**: Must start and end with "/"
5. **Plugins**: Must be installed in node_modules
6. **Sidebar**: Must reference valid docs directory

## State Transitions

N/A - Static configuration, no state changes during runtime.

## Relationships

- Configuration → Sidebar (references sidebars.js)
- Configuration → Custom CSS (references src/css/custom.css)
- Configuration → Preset (uses @docusaurus/preset-classic)
- Configuration → Plugin (uses docusaurus-plugin-sass)
